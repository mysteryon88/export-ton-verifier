# Export TON Verifier

## Installation

```bash
git clone https://github.com/mysteryon88/export-ton-verifier.git
cd export-ton-verifier
npm install
```

## Usage

```sh
node  ./export-ton-verifier/src/cli.js ./circuits/test.zkey ./contracts/verifier.fc
```
